◆ ParamID

enum ParamID

Parameter IDs.

Parameter IDs here match AT S-register numbers, so change them with extreme care. Parameter zero cannot be written by AT commands.

Note that this enumeration is used to initialise the parameter_names array, and parameters not listed in that array will not be visible by name.

When adding or removing a parameter here, you must also update:
