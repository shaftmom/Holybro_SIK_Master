Detailed Description

Storage for program parameters.

Parameters are held in a contiguous array of 16-bit values. It is up to the caller to decide how large a parameter is and to access it accordingly.

When saved, parameters are copied bitwise to the flash scratch page, with an 8-bit XOR checksum appended to the end of the flash range.
